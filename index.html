Linux Shortcuts:

Se déplacer sur le bureau :

  • Alt+F4 Fermer la fenêtre actuelle ★
  • Alt+Tab Changer de fenêtre rapidement ★
  • Ctrl+Alt+Flèches Basculer entre les bureaux ★
  • Ctrl+Alt+L Verrouiller l'écran ★

  • Ctrl+x Couper ★
  • Ctrl+c Copier ★
  • Ctrl+v Coller★
